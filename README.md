# Analog Lab Experiments

This repository contains the LTspice simulations and documentation developed as part of the Analog Electronics Laboratory coursework. The objective of these experiments was to understand the design, analysis, simulation, and integration of analog building blocks used in a Class-D Audio Amplifier system.

## Experiments

### 1. Ramp Generator

Designed and simulated a ramp generator circuit used to generate a periodic triangular/ramp waveform. This waveform serves as the reference signal for PWM generation.

### 2. Single-Ended to Differential Input Converter and PWM Modulator

Implemented a circuit to convert a single-ended input signal into a differential signal and generated Pulse Width Modulation (PWM) signals by comparing the input signal with the ramp waveform.

### 3. H-Bridge Driver and Integration

Designed and analyzed an H-Bridge driver stage for efficient switching operation. Integrated the PWM output with the switching stage to drive the load.

### 4. Band-Pass Filter

Designed a band-pass filter to recover the amplified audio signal and suppress unwanted switching-frequency components generated by the PWM stage.

### 5. Adder

Implemented an analog adder circuit using operational amplifiers to perform signal summation and conditioning as required by the system design.

### 6. Top-Level Integration

Integrated all previous modules:

* Ramp Generator
* PWM Modulator
* H-Bridge Driver
* Band-Pass Filter
* Adder

Performed complete system-level simulation and verification to analyze the overall functionality and performance of the Class-D amplifier architecture.

## Tools Used

* LTspice
* Analog Circuit Analysis
* Operational Amplifiers
* Passive Components (Resistors, Capacitors)
* PWM Techniques
